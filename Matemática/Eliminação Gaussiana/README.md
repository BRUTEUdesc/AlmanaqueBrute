# Eliminação Gaussiana

Método de eliminação gaussiana para resolução de sistemas lineares.
* Complexidade de tempo: O(n³).


Dica: Se os valores forem apenas 0 e 1 o algoritmo gauss_mod2 é muito mais rápido.
